  • Approximation for Φ(x)\Phi(x)Φ(x)
    The shapes of CDF for logistic and normal distribution are similar——both are bell shaped. We can use the logistic CDF, which has a simpler formula for calculation, to approximate the CDF Φ(x)\Phi(x)Φ(x) for standard normal distribution.
    Obviously, their μ\muμ should be same, both zero. Thus, there are only one parameter left for logistic CDF to determine——the sss. Let γ=1s\gamma = \frac{1}{s}γ=s1​, we try to find the gamma that minimizes the difference between the two CDFs

    Using the generalized reduced gradient algorithm, the parameter γ\gammaγ is determined by minimizing the maximum deviation between the normal and logistic CDFs.

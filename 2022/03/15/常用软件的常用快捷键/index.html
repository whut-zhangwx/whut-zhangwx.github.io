常用软件的常用快捷键

发表于2022-03-15|更新于2022-08-18|计算机
|字数总计:432|阅读时长:1分钟|阅读量:

Chrome

命令 快捷键
打开新的标签页 Ctrl+T
打开新的窗口 Ctrl+N
在隐身模式下打开新窗口 Ctrl+Shift+N
重新打开上次关闭的标签页 Ctrl+Shift+T
切换下一个标签页 Ctrl+Tab
切换上一个标签页 Ctrl+Shift+Tab
打开当前网页的源代码 Ctrl+U
打开开发人员工具 Ctrl+Shift+J
横向滚动 Shift+鼠标滚轮
页内查找 Ctrl+F
显示或隐藏书签栏 Ctrl+Shift+B

Typora

命令 快捷键
缩进 / 取消缩进 Ctrl+]/[
查找 / 替换 Ctrl+F/H

NodePad++

命令 快捷键
注释 / 取消注释 Ctrl+K / Ctrl+Shift+K
Ctrl+Q / Ctrl+Q
缩进 / 取消缩进 Tab / Shift+Tab
查找 / 替换 Ctrl+F/H

VsCode

命令 快捷键
显示命令面板 Ctrl+Shift+P / F1
注释 / 取消注释 Ctrl+/ / Ctrl+/
缩进 / 取消缩进 Ctrl+]/[
显示集成终端 Ctrl+~
查找 / 替换 Ctrl+F/H

TeXstudio

命令 快捷键
注释/取消注释 Ctrl+T / Ctrl+T/U
查找 / 替换 Ctrl+F/R

PyCharm

命令 快捷键
注释 / 取消注释 Ctrl+/ / Ctrl+/
缩进 / 取消缩进 Tab / Shift+Tab

IDEA

File->Settings->Keymap中修改快捷键

命令 快捷键
运行 默认Shift+F10
修改Ctrl+Alt+N
保存 Ctrl+S
注释 / 取消注释 Ctrl+/ / Ctrl+/
缩进 / 取消缩进 Tab / Shift+Tab
查找 / 替换 Ctrl+F/R
删除当前行 Ctrl+Y
复制当前行 Ctrl+D
补全代码 Alt+/
快速对齐 Ctrl+Alt+L
快速定位方法 Ctrl+B
显示类层级 Ctrl+H
自动分配变量名 .var
快速生成构造方法 笔记本键盘Alt+Fn+F12
标准键盘 Alt+Insert
鼠标右键 Generate
自动导入 Alt+Enter
在File->Settings->Editor->General->Auto Import->Java中勾选
✓▢ Add unambiguous imports on the fly
✓▢ Optimize imports on the fly

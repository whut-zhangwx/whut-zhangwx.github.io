😂 简要介绍

Markdown 是一种轻量型标记语言, 是一种语法. 以 .md 结尾的文本文件就是 Markdown 文件. 相较于 Word, 它更加像是 HTML 语言或是 $\LaTeX$, 并不是最淳朴的那种”所见即所得”. 它处处透露着一种极简主义. 高效简洁清晰的同时, 又很简单. 看起来舒服, 语法简单, 尤其在处理纯文本上有很大的优势.

它相较于 Word, 兼容性非常高, 可以跨平台使用, 不用担心奇奇怪怪的版本兼容问题. 同时, 有许多网站都支持或正在使用 Markdown 语法. 如 Github (等一系列代码托管平台), StackOverflow(等答疑平台), 简书, 语雀 (等一系列笔记平台).

Typora

Typora 官方中文站

Typora 是被最广泛用于写 Markdown 的文本软件

2021年11月，Typora 已经发布正式版Typora1.0版本，并且变为收费软件，价格为￥89一次购买制, 可以用在3台设备上，支持后期免费更新.

最后一个测试版/Beta版本是Typora 0.11.18，官方历史版本的下载链接 Typora 历史版本下载页(已失效)

Typora Beta版过期解决

按Windows+R打开运行窗口，输入regedit，点确定，打开注册表，地址栏粘贴Computer\HKEY_CURRENT_USER\Software\Typora，然后在Typora上右键，点Permissions，选中Administrtors，把权限全部勾选Deny。

在Typora中File->Preference->General->Update中去勾选▢ Check updates automatically

🍭 基础教程

标题有 * 表示该为扩展语法, 仅在 Typora 或 添加了扩展的 VS Code 本地生效, 在大多数平台上并不认可.

0. 关于半角标点

建议在文本中使用半角的英文标点 ,.?!()[]:;"", 并在每个半角标点后加上空格, 符合英文标点的书写规范.

微软自带输入法, 在使用中文输入法时按下 Ctrl + ., 可以切换标点的全角与半角. 这样即可在中文输入状态下使用半角标点.

1. 标题

Markdown 语法本身可以支持6级标题, 但为了增强文档的易读性, 建议最多使用4级标题

在 Typora 中File->Preference->General->Update->Syntax Preference 中勾选▢ Strict Mode(严格模式), 在严格模式下, 表示标题的#后必须加空格, 标题才能生效, 没有空格不能生效.

这样使得Typora所写的文本也可以很好地兼容其他文本编辑器.

11. *上标 [用 “^” 包围]

(注意: 此为扩展语法)

在File->Preference->Markdown->Syntax Support中勾选 ▢ Superscript (e.g: X^2^), 重启Typora即可生效

12. *下标 [用 “~” 包围]

(注意: 此为扩展语法)

在File->Preference->Markdown->Syntax Support中勾选▢Subscript (e.g: H~2~O), 重启Typora即可生效

H~2~O 是剧毒的!

由于 * 与 _ 均会自动补全, 所以我觉得 - 最为方便

(注意: 英文输入为扩展语法)

这个功能唯一的要求就是英语水平要高, 或者大概记得各个 Emoji 的英文名.

:drooling_face:
:clown_face:

对于其余普通的 Markdown 文本编辑器, 可以直接将 Emoji 表情复制进来😅🤤🤡, 这是直接硬编码的

用好这个功能可以让你的文本非常的可爱! 这里分享一个可以复制全Emoji的网站, 非常好用!

🔥 进阶教程

1. 目录 [自动生成]

若使用 VS Code 搭配 Markdown All in One 扩展, 可在 VS Code 的命令面板 (即 VS Code Command Palette) 输入 Create Table of Contents 自动生成目录, 且可在扩展设置中细调目录参数.

2. 内联 HTML 代码 [用 “<> “ 包围]

只要你会写, 你完全可以把 Markdown 当作 HTML 来写.

同时, .md 文件可以直接导出成一个网页.

下划线可以选中想要下划的文字按下 Ctrl + U.

E.G.

我不会 HTML 呜呜呜... 浇浇我
简单的文字居中也可以这样

我差点忘了还有下划线这东西…

3. 内联 $\LaTeX$ 公式 [用 “$” 包围]

(注意: 部分编译器会不识别部分符号)

用 并敲回车即生成公式块.

E.G.

$\LaTeX$ 是最好用的论文排版语言! 不信你看!

$a^n+b^n=c^n$

4. *网络图床

(这是学长给我的网络图床教程, 我并未实践过…)

分享一个 Typora 搭配腾讯云COS/阿里云OSS图床的例子. 新用户免费试用6个月, 另外还可选择七牛云或者路过图床.

5. *Typora 的常用快捷键

按键 效果 按键 效果
Ctrl + D 选中当前词 Ctrl + L 选中当前句/行
Ctrl + E 选中当前区块 Ctrl + F 搜索当前选中
Ctrl + B 加粗当前选中 Ctrl + H 替换当前选中
Ctrl + I 倾斜当前选中 Ctrl + U 下划当前选中

还有一些不常用的/三键的快捷键不在此列出.

6. *Typora 的主题样式与检查元素

Markdown 在编译后约等于 HTML. 而 Typora 的正常视图就是编译后的 Markdown, 故Typora的主题样式本质就是 CSS 文件.

可以下载各种好看的主题给 Typora换上, 同时也可以自己调整对应的 CSS 文件, 或者自己手搓.

在 Typora 设置中开启 调试模式 后即可在正常视图右击打开 检查元素, 在其中就可以完全将 Markdown 文件当成 HTML 来编辑.
